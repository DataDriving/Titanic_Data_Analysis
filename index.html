A Visual and Satistical Analysis of the Titanic Survivor Data¶

Introduction¶

The following analysis of the Titanic Data aims to discover any correlation between certain demographics/location parameters and survival rates. The data covers 891 passengers with information related to their passenger class, name, gender, age, number of siblings and spouses, number of parents and children, ticket ID, fare price, cabin, and embarkment location. Using the data, the following questions can be posed:

Demographic based survival questions:

  • Were women more likely to survive than men?
  • Were children more likely to survive than adults?
  • Were families more likely to survive than single adults?
  • Were first class passengers more likely to survive than second or third?

Location based survival questions:

  • Were passengers from a certain embarkment location more likely to survive?
  • Were passengers from specific Cabin groups (A-G) more likely to survive?
  • Were passengers with lower numerical Cabin locations more likely to survive?
  • Were passengers with higher fares in each Cabin group more likely to survive?
  • Were passengers with higher fares in each Passenger Class more likely to survive?

Data Wrangling¶

There will be 4 new columns created to make finding trends and correlations more manageable:

  • AgeCat - passengers put into specific Age Categories. There will be better insight to compare passengers in different age groups while also giving the null values a category so that it takes away any issues when comparing data.
  • SingFam - passengers will be put into two categories - single or family - depending on their values in SibSp or Parch. This will also make it easier to find correlation between survivors and non-survivors
  • The Cabin values are difficult to process as is, so two new columns will be created:
    • CabLet - Passengers with Cabin values will be grouped by the first letter in their cabin location
    • CabNum - Cabins that have a single Cabin location will have their numerical value parsed

def categorize_letter(text):
    #Make both NaN and single character entries undefined
    if pd.isnull(text) or len(text) == 1:
        return 'undefined'
    else:
        #Entries with a single character prefacing an actual cabin value
        if text[1] == ' ':
            return text[2]
        else:
            try:
                #All other entries should have the Cabin Letter as the first
                return text[0]
            except:
                #Error handling
                return 'error'

def categorize_number(text):
    if pd.isnull(text) or len(text) == 1:
        return -1
    else:
        #Entries with a single character prefacing an actual cabin value
        if text[1] == ' ':
            #Takes the numbers following the second entry
            return text[3:]
        #Entries with multiple Cabin values
        elif len(text) > 3 and text[3] == ' ':
            #Takes the first Cabin's numbers (No 3 digit Cabins possible)
            return text[1:2]
        #Entries with only one Cabin value
        elif ' ' not in text:
            return text[1:]
        else:
            #Error Handling
            return -2
        
#Entering the two series into the dataframe

Visual interpretations¶

From the questions posed in the introduction:

  • Were women more likely to survive than men?

    It does seem that women were significantly more likely to survive than men.

  • Were families more likely to survive than single passengers?

    It looks like half of the passengers with families survived, while only a third of single passengers did the same.

  • Were children more likely to survive than adults?

    Although it's difficult to tell with each of the categories under different scales, it looks like infants were more likely to survive than the other categories, but children and youth seemed to be no different than young and middle adults. The older adults and elderly look like they were more likely to not survive.

  • Were first class passengers more likely to survive than second or third?

    First class passengers were more likely to survive than Second and Third. Second class passengers were also more likely to survive than Third.

  • Were passengers from certain Embarkment locations more likely to survive?

    It's hard to tell, but it looks like passengers from 'C' survived the best, and from 'Q' the worst.

  • Were passengers from specific Cabin groups more likely to survive?

    Visually, it doesn't look like any particular group is an outlier, or if there is a trend among the groups.

  • Were passengers from lower numerical Cabin locations more likely to survive?

    It appears for most Cabin groups, the survivors were slightly more likely to have lower numerical values

  • Were passengers with higher fares in each Cabin group more likely to survive?

    For Cabin group B only there appears to be a large difference in fares paid between survivors and non-survivors.

  • Were passengers with higher fares in each passenger Class more likely to survive?

    For First class passengers it looks like survivors had a higher fare than non-survivors, but no discernible difference between survivors' and non-survivors' fares for Second and Third class passengers.

With these insights, a few more questions can be posed for further analysis:

  • Were men with families more likely to survive than single men?
  • Were women traveling alone less likely to survive than those with family on board?

  • Were certain embarkment locations correlated to certain passenger classes?

Statistical Inference¶

To keep visual interpretations from being misleading, there are 3 tests that can verify the analysis:

  • The Chi Square test can be used for comparing two sets of categorical data (All of the bar charts)
  • The T-test can be used for comparing numerical data of two categories (All of the box plots)
  • The Linear Regression test can be used to find trends between two sets of numerical data (The scatter plot)

#Chi Square test
def chi2test(question, desc1, desc2, crosstab):

    #Performs Chi Square on the crosstab data
    chi2, pval, df, expval = chi2_contingency(crosstab)
    
    #Accepts or rejects the null hypothesis of
    #independence between categories
    if pval < 0.05:
        ans=('REJECTED with a 95% confidence level',
             'dependent')
    else:
        ans=('NOT REJECTED with a 95% confidence level',
             'independent')
    
    
    str1 = 'Cross-tabulation between {d1} and {d2}:'.format\
            (d1=desc1,d2=desc2)
    #Show the table of the initial data
    table1 = crosstab
    str2 = 'Expected values from the chi-square test:'
    #Show the table of the expected values
    table2 = pd.DataFrame(np.around(expval, decimals=1),
                    columns=list(crosstab.columns.values),
                    index=list(crosstab.index.values))
    #Show the test statistics
    str3 = '\nchi-square statistic = {c2:.3f} \
            \n  degrees of freedom = {deg} \
            \n             p-value = {pv:.3E} \
            \n     null hypothesis = {hyp} \
            \n\n##### Relationship between {d1} and {d2} is likely to be {hyp2} \
            \n'.format(c2=chi2,deg=df,pv=pval,d1=desc1,
             d2=desc2,hyp=ans[0],hyp2=ans[1])
    
    #Print report
    display(Markdown('### '+question))
    display(Markdown('\n###### '+str1))
    display(table1)
    display(Markdown('\n###### '+str2))
    display(table2)
    display(Markdown(str3))
    display(Markdown('---'))
    
    return

#T-test for difference in mean
def ttest(question, cat1, cat2, df):
    #Build a summary table to show all of the comparisons
    collabel = ['n0','n1','x0','x1','s0','s1','T-stat',
                'p-value','null hyp']
    rowlabel = df[cat1].unique().tolist()
    rowlabel.sort()
    
    #Initializes an array of zeros with the
    #right number of cells to fit the table
    vals = np.zeros((len(rowlabel),len(collabel)))
    
    #Writes descriptive statistics into table for each
    #comparison between surviving and non-surviving passengers
    for i, s in enumerate(rowlabel):
        vals[i,0] = df[(df[cat1]==s)&(df['Survived']==0)][cat2].count()
        vals[i,1] = df[(df[cat1]==s)&(df['Survived']==1)][cat2].count()
        vals[i,2] = df[(df[cat1]==s)&(df['Survived']==0)][cat2].mean()
        vals[i,3] = df[(df[cat1]==s)&(df['Survived']==1)][cat2].mean()
        vals[i,4] = df[(df[cat1]==s)&(df['Survived']==0)][cat2].std()
        vals[i,5] = df[(df[cat1]==s)&(df['Survived']==1)][cat2].std()
        #Runs the t-test to return the test staistic and the p value
        t = ttest_ind_from_stats(vals[i,2],vals[i,4],vals[i,0],
                                 vals[i,3],vals[i,5],vals[i,1],
                                 equal_var=False)
        
        #Writes the final statistical values into the table
        vals[i,6] = t[0]
        vals[i,7] = t[1]
        #A two-tailed t-test is assumed since the difference in the
        #means is being tested in either direction
        if t[1] < 0.025:
            ans = 1
            #Since this NumPy array can only hold numerical values,
            #1 will represent a rejection of the null.
        else:
            ans = 0 #0 represents that the null cannot be rejected
        vals[i,8] = ans
    #Builds the Pandas table of the T-tests in each category
    vals = np.around(vals,decimals=3)
    table = pd.DataFrame(vals,columns=collabel,index=rowlabel)
    
    #Prints report
    display(Markdown('### '+question))
    display(Markdown('A value of 1.0 under null hyp indicates that the null\
    \nhypothesis was rejected under a 95% confidence level'))
    display(table)
    display(Markdown('---'))
    return

def linreg(question, cat1, cat2, df):
        #Runs the linear regression test
        lr = linregress(df[cat1].tolist(),df[cat2].tolist())
        #Builds report with the test statistics
        str1 = '\n    slope = {slp:.3f}\
                \nintercept = {cpt:.3f}\
                \nr-squared = {rvl:.3f}\
                \n  p-value = {pvl:.3E}\
                \n  std err = {ser:.3f}\
                '.format(slp=lr[0],cpt=lr[1],rvl=lr[2]**2,
                         pvl=lr[3],ser=lr[4])
        #Accepts/rejects the null hypothesis under a 95% confidence level
        if lr[3] < 0.05:
            ans = 'some significant correlation between {cat1}\
            and {cat2}'.format(cat1=cat1,cat2=cat2)
        else:
            ans = 'no significant correlation between {cat1} \
            and {cat2}'.format(cat1=cat1,cat2=cat2)
        str2 = '\n\n##### With a p-value of {pvl:.3E}, there is\
        {ans} under a 95% confidence level.'.format(pvl=lr[3],ans=ans)
        #Prints Report
        display(Markdown('### '+question))
        display(Markdown(str1))
        display(Markdown(str2))
        display(Markdown('---'))
        return

Conclusions and Limitations¶

From Chi Square testing¶

Comparing the actual value vs the expected value, along with the p-value:

  • Women were overwhelmingly more likely to survive than men.
  • Families were more likely to survive than single passengers.
  • Younger age categories were more likely to survive than older categories.
  • First class passengers were more likely to survive than second, and second more likely than third.
  • Passengers from the 'C' embarkment location were more likely to survive, and 'S' less likely to survive.
  • Passengers from 'B', 'D', and 'E' Cabin groups were more likely to survive than expected, but the null hypothesis would not have been rejected with a confidence level higher than 95% with a p-value of .0411. So while survival is dependent on Cabin Groups, it's not as dependent as it is on other parameters.
  • Men with family members on board were more likely to survive than men without family. This might be because males with families skew slightly younger than single males, which could cause some correlation to survival since younger passengers were more likely to survive:

From T-testing between two sample means¶

  • There was no statistical difference between Cabin Number of survivors and non-survivors among any of the Cabin Groups.
  • Within Cabin Group 'B' and 'G' only, there was a statistical difference between the fares paid by survivors and non-survivors. Survivors with undefined Cabins had also paid higher fares than non-survivors, but this is likely because without the Cabin groups defined, there are passengers from different classes all represented here (Higher fares for First Class passengers who were more likely to survive in general).
  • Within First Class, survivors were statistically paying a higher fare than non-survivors, but no difference in fares for Second or Third Class.

From Linear Regression¶

  • With an extremely low r-squared value and a p-value outside the 95% confidence interval, there does not seem to be a linear relationship between Cabin Number and Fare price

Limitations of Analysis¶

  • The lack of age entries from all the data limits the understanding of its relationship to survivorship. A common assertion in popular media on passenger survival was that women and children were to go first, but survival was not as dependent on Age Categories than it was on Passenger Class. Since there were many missing entries on age, it's difficult to say conclusively that age was less important than passenger class.
  • It is still difficult to draw any conclusions on survival based on location about all of the passengers since so many entries did not have a Cabin value present The location based questions could definitely have been more in depth if the dataset had more entries on Cabins. It may have also been helpful to have a dataset tying Cabins (or a group of Cabins) to a parameter showing the distance to the nearest lifeboat.
